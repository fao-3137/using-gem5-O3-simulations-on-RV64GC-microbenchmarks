# using-gem5-O3-simulations-on-RV64GC-microbenchmarks

这是一个代码脚本库以及对应代码跑出来的数据，有四组实验，区分“预测器降低失败频率”与“结构放大单次失败破坏”的贡献，设计四组互补对照实验：
（1）固定结构点的预测器档位扫描：在给定 (W, pipe_scale) 下改变预测器类别与其配置，形成多个 MR 点，刻画 MR 改善如何转化为 IPC与PLR 的提升，并据此估计 BSF 的程序差异。
（2）固定预测器档位的结构扫参：在固定预测器族与档位下扫描 W 与 pipe_scale，使 MR 尽可能保持同量级，从而把 PLR与BSF 的变化主要归因于结构放大效应，并比较“宽度”与“前端深度/恢复链路尺度”的主效应量级。
（3）固定结构点的跨程序对比：在同一结构点上对 10 个微基准重复(1)，对不同分支密度与可预测性类型下的 BSF 差异进行比较，并结合动态分支次数（BrCount）与误预测压力（BrCount×MR）等解释变量进行归因。
（4）在真实开源测试集GAPBS下的表现

采用gem5作为核心实验平台，基于RISC-V64GC（RV64GC） ISA编译自定义微基准测试程序，并以乱序执行（Out-of-Order,O3）CPU模型作为微架构载体。
